# Oh Shit!
Oh Shit! is a four person "free-for-all" card game (no teams) played with a standard 52 card deck.  
The general game play is similar to Euchre.  A dealer is randomly determined, but be warned; being 
the deal is disadvantageous.  The game consists of at least 13 rounds (a "short" game) or 25 
(a "long" game).

## Why "Oh Shit!"
Well, because if each player bids accurately, the optimal bid for the dealer will be the one bid 
disallowed by rule (usually causing the dealer to say "Oh Shit!")

## Where did I learn this game
David Lash, the father of one of my best friends, taught it to us while attending Purdue...in my 
buddy's house on 4th street.

## Why would you like this game?
If you like Euchre, but find it almost _too_ easy and would prefer a game that requires more 
attention and has more variability, this is the game for you.  My friends and I have played this 
game for years at houses, bars, tailgates...anywhere.  It's actually _very_ easy to play.  One game 
in and you'll have the hang of it!
